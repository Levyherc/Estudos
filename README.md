## 🌟 Projetos em Destaque

### 💡 Projeto Integrador – Proz + AWS
Uma aplicação web completa desenvolvida em equipe, com foco na criação de interfaces responsivas e integração com APIs externas. Este projeto foi realizado como parte do curso de Front-End com foco em Cloud Computing da Proz Educação em parceria com a AWS.

🔗 [Acesse o Projeto Online](https://levyherc.github.io/Projeto-Integrador-Proz-AWS/5-integrador-4.0-projeto-final/index.html)  
📁 [Repositório GitHub](https://github.com/Levyherc/Projeto-Integrador-Proz-AWS)

#### 🚀 Sobre o Projeto

O projeto tem como objetivo simular uma plataforma informativa sobre um tema atual, utilizando ferramentas de desenvolvimento front-end modernas. A proposta inclui:
- Interface interativa e responsiva.
- Integração com APIs para obtenção de dados dinâmicos.
- Apresentação de conteúdo relevante, com foco em usabilidade.

#### 🛠️ Tecnologias Utilizadas
- **HTML5**  
- **CSS3**  
- **JavaScript (ES6+)**  
- **Bootstrap**  
- **Git & GitHub**  
- **Hospedagem via GitHub Pages**
